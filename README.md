# FTP File Transfer System (Java)

A Java implementation of a simplified FTP client/server architecture based on RFC 959, built with low-level socket programming.

## Features
- User authentication (username/password)
- Directory navigation
- File upload & download (RETR, STOR)
- Command parsing (LIST, QUIT, etc.)
- Access control and error handling
- Modular client–server design

## Technologies
- Java
- TCP Sockets
- RFC 959 Concepts

## How to Run
1. Clone the repository.
2. Compile the Java code.
3. Run the `Server`.
4. Run the `Client` and execute FTP commands.
